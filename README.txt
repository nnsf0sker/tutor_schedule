# tutor_schedule
 A small console app that places lessons in timetable.
 Маленькое консольное приложение составляющее расписание учеников.
 
 
 
 Программа берёт на вход в локальной папке файл input.txt, который должен иметь следующую структуру:
 --------begin:
 Name1 TimePoint11 TimePoint12 ... TimePoint1N_1
 Name2 TimePoint21 TimePoint22 ... TimePoint1N_2
 ...
 NameK TimePointK1 TimePointK2 ... TimePoint1N_K
 ________end;
 
 То есть в файле K строк (где K - количество учеников), каждая из которых состоит из слов, разделёных пробелом. Первое слово является именем ученика (далее ChildId), каждое последующее слово является названием временного промеждутка (далее TimePoint), в которое данный ученик может заниматься.
 Любые два одинаковых названия TimePoint'а обозначают один и тот же промежуток. Если у ученика два раза написан один и тот же TimePoint, это эквивалентно одному написанию этого TimePoint'а.
 Любые два одинаковых ChildId в разных строках будут обрабатываться, как два разных ученика (в связи с этим возможна неоднозначность ответа, поэтому разных учеников лучше называть разными именами).
 
 Программа выводит в консоль построчечно пары:
 Namei_1 TimePointj_1
 Namei_2 TimePointj_2
 ...
 Namei_m TimePointj_m
 
 обозначающие, что ученика с ChildId==Namei_1 нужно поставить в расписании на TimePointj_1, причём такая расстановка избежит конфликтов между разными учениками (все они будут заниматься в разное время).
 Критерием корректности решения является максимальность количества выведенных строк (то есть другим способом расставить бóльшее количество учеников нельзя).
